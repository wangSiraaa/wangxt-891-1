# 🛠️ 社区共享工具借还台

一个基于浏览器本地存储的社区工具共享管理应用，支持居民借用、管理员管理、维修志愿者维护等完整流程。

## ✨ 功能特性

### 👥 角色系统
- **居民**：浏览工具、借用工具、归还工具、查看借用记录、排行榜
- **管理员**：锁定/解锁工具、查看所有记录、管理维修、查看操作日志、重置数据
- **维修志愿者**：处理维修任务、开始/完成维修、查看维修记录

### 📦 主数据
- 工具卡片（名称、分类、描述、押金、状态、累计借用次数）
- 工具分类（电动工具、手动工具、园艺工具、清洁工具、测量工具、其他）
- 押金状态与管理
- 借用篮
- 借用记录
- 归还备注
- 维修标记
- 损坏照片占位
- 逾期提醒
- 借用排行榜
- 操作日志

### 🔄 主流程
1. **工具筛选**：按分类、状态、关键词搜索
2. **加入借用篮**：选择需要借用的工具
3. **押金确认**：确认押金金额和借用期限
4. **借出**：确认借出，工具状态变更
5. **归还**：填写归还状态，如有损坏需登记
6. **状态检查**：工具状态实时更新
7. **损坏登记**：损坏工具自动进入维修流程
8. **维修完成**：维修志愿者处理后恢复可借状态
9. **排行榜刷新**：借用数据实时更新排行榜

### 🚧 业务边界规则
- ✅ 损坏或维修中的工具不能借出
- ✅ 归还时必须填写工具状态说明
- ✅ 选择损坏时必须填写损坏处理备注
- ✅ 同一居民有逾期未归还记录时只能预约不能借出
- ✅ 同一工具不能被重复加入借用篮
- ✅ 管理员锁定工具后普通居民不能操作
- ✅ 所有状态共享同一套数据源，列表/详情/按钮状态一致

### 💾 本地持久化
- 所有数据保存在浏览器 `localStorage` 中
- 刷新页面后借用记录、维修标记、排行榜仍可复查
- 支持一键重置数据

## 🧪 Smoke 测试验证点

Smoke 脚本至少验证以下核心功能：

1. **刷新页面后借用记录仍保留**
   - 数据通过 localStorage 持久化存储
   - 状态变更后自动保存

2. **损坏工具借出按钮禁用**
   - 损坏状态的工具卡片显示"已损坏"标签
   - 加入借用篮按钮不可用
   - 工具详情页提示损坏信息

3. **归还损坏工具未填备注无法提交**
   - 归还时必须填写状态说明
   - 选择"有损坏"时必须填写损坏处理备注
   - 未填写时提交按钮禁用并给出提示

## 🎮 使用说明

### 角色切换
在页面顶部的角色选择器中可以切换不同身份，体验不同角色的功能。

### 居民操作流程
1. 浏览工具列表，使用筛选器查找需要的工具
2. 点击工具卡片查看详情
3. 点击"加入借用篮"将工具加入
4. 进入借用篮页面，确认工具和押金
5. 选择借用期限，点击"确认借出"
6. 使用完毕后，在"借用记录"中点击"归还"
7. 填写工具状态，如有损坏请详细描述
8. 提交归还，押金退还

### 管理员操作
- 锁定/解锁工具，控制工具是否可借用
- 查看所有借用记录
- 查看操作日志，追踪所有操作
- 重置所有数据到初始状态

### 维修志愿者操作
- 查看待维修的工具列表
- 点击"开始维修"接手维修任务
- 维修完成后点击"维修完成"
- 工具自动恢复可借用状态
